# SynTactic
ESaaS Fall '23 Project

# Installation

- Clone repo
- Run `bundle install`
- Run `npm install`
- Run `rake db:create`, `rake db:migrate`, then `rake db:seed`

# Running

- Start the Rails server and esbuild with one command `./bin/dev`
- Hit [http://localhost:3000/home/](http://localhost:3000/home/)
